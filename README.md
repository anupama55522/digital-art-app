# 🎨 Digital Art Gallery App (Android)

A mobile application designed to connect artists and customers through a digital platform for showcasing, exploring, and purchasing artworks. The system ensures a smooth experience with role-based access for admins, artists, and customers.

## 📱 Project Overview

The **Digital Art Gallery App** is an Android-based platform that enables:

- Artists to upload and manage their artworks  
- Customers to browse, search, and purchase artworks  
- Admin to monitor, verify, and manage users  

The application focuses on usability, real-time updates, and secure data handling.

---

## ✨ Key Features

### 👨‍💼 Admin Module
- View registered users  
- Verify artists through certificates  
- Delete users (artists/customers)  

### 👩‍🎨 Artist Module
- Register with certification  
- Upload and manage artworks  
- Edit or delete artwork details  

### 🛍️ Customer Module
- Register and login  
- Search artworks  
- Purchase and request custom artwork  
- Cancel orders within a limited time  

### 🔐 Authentication System
- Secure login for users and admin  
- Session management with logout functionality  

### 👤 Profile Management
- View profile  
- Update profile details  
- Delete account  

### 🖼️ Artwork Management
- Add artwork (title, description, price, etc.)  
- Edit artwork details  
- Delete artwork  

### 🔍 Search System
- Search artworks by name  
- Display relevant results dynamically  

### 💳 Purchase System
- Buy artworks  
- Customize artwork requests  
- Real-time availability updates  
- Automatic hiding of sold items  

### 💬 Feedback System
- Add feedback  
- View feedback  
- Update or delete feedback  

---

## ⚙️ System Architecture (Overview)

The application follows a structured flow:

- User inputs are processed through the frontend  
- Backend handles logic and validation  
- Data is stored and retrieved using SQLite database  
- Real-time updates ensure data consistency
